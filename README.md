### 必要なページは？
メインページ

ファイル：home.tsx

活動スケジュール
これから行われるイベントの宣伝ページ
ファイル：schedule.tsx

クラブ紹介ページ
この部活のこと（例えば、競技用ドローン、telloなど）を紹介するページ
ファイル：about.tsx
  
ブログ
活動記録・その他を載せるページ
ファイル：blog.tsx

お問い合わせ
問い合わせフォーム
ファイル：contact.tsx

アクセス
Google map（先生方と相談）
ファイル：access.tsx

（空撮動画・画像）

ファイル：media.tsx
